  1. Dans ton bot GREENROAD : onglet 📜 Historique → bouton "📤 Exporter JSON vers Telegram"
  2. Le bot envoie un fichier greenroad_trades_YYYYMMDD.json sur ton Telegram
  3. Depuis Telegram sur ton iPad : sauvegarde le fichier (ex: dans Fichiers)
  4. Ici, clique "📁 Choisir le fichier JSON" et sélectionne-le
  5. Les stats et graphiques s'affichent automatiquement
