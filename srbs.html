📖 Procédure d'export depuis le bot

  1. Dans ton bot GREENROAD : onglet 📜 Historique → bouton 📤 Export Telegram
  2. Choisis la période (7j / 30j / 90j / Tout) → Envoyer
  3. Le bot envoie greenroad_trades_YYYYMMDD_HHMMSS.json sur Telegram
  4. Depuis Telegram sur ton iPad : télécharge le fichier dans Fichiers
  5. Ici → 📁 Choisir le fichier JSON → l'analyse s'affiche
