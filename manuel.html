🚀 Demarrage rapide

Premiere fois avec GREENROAD ? Suis ces 5 etapes.

1. Mode DEMO d'abord

  • Va dans Config -> Risque et verifie que 'Mode DEMO' est coche.
  • Ca te permet de tester sans risque : aucun ordre reel n'est envoye.
  • Fais au moins une journee complete en DEMO avant de passer en LIVE.

2. Choisir une strategie

  • Onglet Strategies : une quinzaine de strategies disponibles, classees par categorie.
  • Pour debuter, SCALP_EMA_RSI ou RANGE_TRADER sont de bons choix.
  • Clique sur une carte pour voir sa description puis 'ACTIVER'.

3. Configurer les parametres risque

  • Config -> Risque : taille de position (5-10% recommande), levier (1 au debut).
  • Active SL et TP avec des valeurs raisonnables (SL 1%, TP 2% par exemple).
  • Active le budget quotidien ($50 max par exemple) pour limiter les pertes.

4. Demarrer

  • Retour Dashboard, clique sur 'DEMARRER'.
  • Observe le bandeau Live Reasoning qui te raconte ce que fait le bot.
  • Les 15 premieres minutes : surveille les logs pour verifier que tout roule.

5. Passer en LIVE (quand tu es pret)

  • Decoche 'Mode DEMO' dans Config -> Risque.
  • Le bot va maintenant envoyer des ORDRES REELS sur Hyperliquid.
  • Commence avec des petites tailles, augmente progressivement selon tes resultats.

⚙️ Config

6 sous-onglets pour tout parametrer : General, Automatisation, Risque, Strategies, Narration, Connexion.

Sous-onglet General

  • Coin : le token trade par defaut (ETH, BTC, SOL, etc.).
  • Timeframe : unite temporelle des bougies (1m, 5m, 15m, 1h, 4h).

Sous-onglet Automatisation

  • Mode Auto-Pilot : active le pilotage automatique.
  • Seuils de confiance : threshold (default 75%) et ask_min (default 60%).
  • Scanner : active le scan multi-tokens + watchlist + nombre max de switches par jour.
  • Re-analyse toutes les X minutes : frequence de reanalyse automatique.

Sous-onglet Risque

  • Taille position (%) : pourcentage du capital mis en jeu par trade.
  • Levier : effet de levier Hyperliquid (1-40). Debuter a 1 ou 2 max.

  • Autoriser SHORT : autorise les positions a la baisse.
  • Kill switches : perte quotidienne max, pertes consecutives max, drawdown max.
  • Budget quotidien en $ : nouveau systeme plus intuitif que le % (defaut $50).

Sous-onglet Strategies

  • Parametres techniques de chaque strategie (EMA fast/slow, RSI seuils, ATR multipliers, etc.).
  • A modifier uniquement si tu sais ce que tu fais.

Sous-onglet Narration

  • Active/desactive le narrateur.
  • Frequence d'observation (3 min par defaut).
  • Story du jour : resume quotidien automatique.
  • Notifications Telegram : choisir quoi envoyer (entrees, sorties, switches, etc.).

Sous-onglet Connexion

  • Wallet principal (compte Hyperliquid).
  • Cle API (private key) chiffree avec mot de passe.
  • Telegram bot token + chat ID.
  • Fees Hyperliquid (taker 0.0432% / maker 0.0144%).

💎 Spot Analyzer

Analyse pure (aucun trade) long-terme de 16 actifs : crypto, indices actions, matieres premieres, forex.

Bouton 'Analyser tous les actifs'

  • Lance le scan de BTC/ETH/SOL/HYPE + S&P500/Nasdaq/Dow + Or/Argent/Brent/WTI/Gaz/Cuivre + EUR/GBP/JPY.
  • Prend 15-30 secondes (telechargement historiques 6 mois pour chaque).
  • Necessite 'pip install yfinance' pour les actifs non-crypto.

Lecture des cartes

  • Une carte par actif, bordure coloree selon le score.
  • Ligne 1 : nom + prix + score/verdict (EXCELLENT / BON / MOYEN / FAIBLE / EVITER).
  • Ligne 2 : variations 24h / 7j / 30j, drawdown depuis ATH, RSI.
  • Ligne SETUP : plage d'entree suggeree + SL + TP + ratio R/R.
  • Narratif en italique : explication en francais naturel.

Score et verdict

  • >= 75 : EXCELLENT MOMENT. Setup tres favorable.
  • 60-74 : BON MOMENT. Setup acceptable.
  • 45-59 : MOMENT MOYEN. Attendre mieux.
  • 30-44 : MOMENT FAIBLE. Risque eleve.
  • < 30 : EVITER. Tres defavorable.

Base de calcul

  • RSI 14 daily, MA200, distance au support/resistance, drawdown depuis ATH.
  • ATR 14 pour dimensionner SL/TP.
  • Score final combine ces metriques.

💡 Conseils

  • C'est une analyse TECHNIQUE pure, pas de fondamentaux ni news.
  • Utilise-la comme aide a la decision pour tes investissements spot.
  • Re-analyse une fois par semaine pour suivre l'evolution.
  • Les setups sont indicatifs, confirme toujours avec ton jugement.

🛡️ Securite (Kill-switch distant)

Systeme de protection a distance : tu peux desactiver TOUS tes bots en quelques minutes via GitHub.

Principe

  • Le bot interroge toutes les 5 minutes un fichier JSON sur GitHub (Gist).
  • Si tu passes le statut sur 'locked' depuis ton telephone -> tous les bots se verrouillent.
  • Utile en cas de vol, fuite du bot, ou urgence (desactivation massive).
  • Aucune dependance autre que GitHub (service fiable, gratuit, permanent).

Badge visuel permanent

  • En haut a droite du Dashboard, a cote des autres badges.
  • 🟢 'PROTECTION ACTIVE' : tout OK, serveur contacte regulierement.
  • 🟡 'ALERTE RESEAU' : plus de contact depuis >24h mais <72h (tolerance).
  • 🔴 'VERROUILLE' : bot bloque par commande distante.
  • Clic sur le badge -> ouvre l'onglet Securite.

Ecran de demarrage

  • A chaque lancement du bot, un ecran de verification s'affiche 3 secondes.
  • Montre : contact serveur + statut recu + confirmation.
  • Rassurant : tu vois que le bot s'est authentifie correctement.

Pour verrouiller a distance

  • 1. Va sur ton Gist GitHub (bouton 'Ouvrir mon Gist' dans l'onglet Securite).
  • 2. Clique 'Edit' en haut a droite.
  • 3. Change "status": "active" en "status": "locked"
  • 4. Clique 'Update secret gist'.
  • 5. Dans les 5 minutes, TOUS tes bots se verrouillent.
  • Pour deverrouiller : repasse 'status' sur 'active'.

Effet du verrouillage

  • Overlay plein ecran rouge avec cadenas.
  • Toutes les positions sont fermees automatiquement au marche.
  • Ordres TP/SL orphelins annules.
  • Bouton DEMARRER bloque (impossible de lancer le trading).
  • Notification Telegram urgente.
  • Les onglets restent consultables (historique, manuel).

Fail-secure 72h

  • Si le bot ne peut pas joindre GitHub pendant plus de 72h -> verrouillage auto.
  • Raison : empeche qu'un voleur contourne en coupant simplement internet.
  • Couper internet = se couper soi-meme.
  • En usage normal tu ne rencontreras jamais ce verrouillage.

Onglet Securite

  • Etat actuel avec couleur (vert/jaune/rouge).
  • Derniere verification + dernier contact reussi + prochaine verification.
  • Bouton 'Verifier maintenant' pour forcer un check immediat.
  • Mode d'emploi inline pour verrouiller/deverrouiller.
  • Historique des 12 dernieres verifications.

💡 Conseils

  • Bookmark l'URL de ton Gist sur ton telephone : c'est ta telecommande.
  • Teste une fois le verrouillage et le deverrouillage pour voir comment ca marche.
  • Garde l'URL du Gist secrete (ne la partage pas, meme si elle n'est pas modifiable sans ton compte).
  • En cas de vol, verrouille immediatement : tes positions seront fermees proprement.

🎓 Concepts cles

Explications des concepts transversaux qui reviennent dans le bot.

Modes de pilotage

  • MANUEL : toute decision demande ta validation (popup).
  • SEMI-AUTO : >=75% execute seul, 60-75% demande validation, <60% ignore.
  • FULL-AUTO : >=75% execute seul, 60-75% REFUSE auto (pas de popup), <60% ignore. Ideal pour vacances.

Patterns chartistes

  • Classiques : Double Top/Bottom, Triangle, Flag.
  • Avances : Head & Shoulders (+ inverse), Divergences RSI haussiere/baissiere.
  • Impact : bloque ou booste un signal selon compatibilite (>=65% confiance).
  • Alerte Telegram auto quand pattern >=80% detecte.

Multi-Timeframe (MTF) Confirmation

  • Avant de trader sur le TF principal, verifie que 1h + 4h confirment la direction.
  • Mode SOUPLE : au moins 1 des 2 doit confirmer.
  • Mode STRICT : les 2 doivent confirmer.
  • Reduit drastiquement les faux signaux.

Sentiment de marche

  • Fear & Greed Index (0-100) : indicateur de panique / euphorie.
  • Funding rates Hyperliquid : cout des positions longues/courtes.
  • Mode informatif par defaut (affiche dans les logs, pas de blocage).

Budget quotidien

  • Plafond de perte en dollars par jour ($50 par defaut).
  • Cumulee en NET (gains compensent pertes).
  • Atteint : refus des nouveaux trades jusqu'a minuit.
  • Alerte anticipee a 70% du budget.

Kill switches

  • Perte quotidienne max en % (kill_max_daily_loss).
  • Pertes consecutives max (kill_max_consec_loss).
  • Drawdown total max (kill_max_drawdown).
  • Spread anormal (kill_max_spread_pct).
  • Cooldown de reactivation (30 min par defaut).

Protections SL/TP

  • Tous les ordres passent avec SL et TP sur Hyperliquid en reduceOnly.
  • Watchdog 30s verifie que les protections sont en place.
  • Cleanup auto des ordres orphelins si aucune position.
  • Confirmation Telegram 🛡️ apres placement reussi.

Narration

  • 200+ templates FR anti-repetition.
  • Observations toutes les 3 min (bandeau + journal).
  • Story quotidienne a 19h (resume journee).
  • Rapport hebdo dimanche 19h sur Telegram.
