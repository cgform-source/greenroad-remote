📖 Comment generer un token ?
1. Va sur github.com/settings/tokens?type=beta
2. Clique Generate new token (fine-grained)
3. Token name : GREENROAD Remote
4. Expiration : No expiration (ou 1 an)
5. Resource owner : ton compte
6. Repository access : Public Repositories suffit
7. Account permissions → Gists → Read and write
8. Clique Generate token en bas
9. COPIE le token (commence par github_pat_)
10. Reviens ici, colle-le, clique Sauvegarder
